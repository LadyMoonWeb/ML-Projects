# ML Projects

В этом репозитории собраны простые проекты для начинающего ML-инженера.

## Список проектов

- **linear_regression_example.py** — линейная регрессия на Python с визуализацией

## Используемые инструменты
- Python
- NumPy
- scikit-learn
- matplotlib
